# レクチャー2.5 詳細設計書

## AIへの指示の基本 ― "伝わるプロンプト"の3つの型

---

## ① レクチャーの位置づけ

レクチャー2.4までで「アカウント作成」「プランの選び方」「セキュリティの安心感」「社内許可の取り方」という"始める前の壁"をすべて取り除いた。本レクチャーは、S2（準備セクション）の締めくくりとして、S3以降の実践パートで毎回使う**「プロンプトの共通言語」**を60秒で頭にインストールする役割を担う。ここで「役割・指示・出力形式」の3語を覚えてもらうことで、S3のPDF抽出で「なぜこう書くと精度が上がるのか」が直感的にわかる状態を作る。次のレクチャー3.1では、この3つの型を実際にClaudeに打ち込み、PDFからデータを抽出する成功体験へと直結させる。

## ④ ライブ実演の操作手順

**該当なし。** 本レクチャーはスライド＋ナレーションのみで構成する。60秒という尺の中で操作を入れると、メッセージがぼやけるため、意図的にライブ実演を排除している。3つの型の実演はS3（レクチャー3.1）で行う。

## ⑤ スライドの指示

```
【スライド1：3つの型タイトル】
- 表示テキスト：
    AIへの指示、3つの"型"
    ① 役割を与える
    ② 具体的に指示する
    ③ 出力形式を指定する
- サブテキスト：なし
- ビジュアル要素：①②③は1行ずつフェードインするアニメーション。各行の左に番号アイコン（丸数字、青色）を配置
- 表示タイミング：0:00〜0:18
- 講師のセリフ（要旨）：「実践に入る前に、AIへの指示のコツを1つだけ覚えてください。たった3つの型です。」→ 1行ずつ読み上げ
```

```
【スライド2：悪い例 vs 良い例】
- 表示テキスト：
    左側（×マーク、赤背景帯）：
      営業リスト作って
    右側（○マーク、青背景帯）：
      あなたは法人営業のアシスタントです。
      以下の企業データから、愛知県内の製造業だけを抽出してください。
      出力は「会社名・住所・電話番号」の3列の表形式でお願いします。
- サブテキスト：右側の各文の横に「←役割」「←具体的な指示」「←出力形式」のラベルを矢印付きで配置
- ビジュアル要素：左右2カラム。左は薄い赤背景、右は薄い青背景。右側の3つのラベルは手書き風フォントで親しみやすく
- 表示タイミング：0:18〜0:40
- 講師のセリフ（要旨）：「左のように曖昧に書くとAIは迷う。右のように3つの型を入れるだけで精度がまったく変わる。」
```

```
【スライド3：まとめ】
- 表示テキスト：
    覚えるのはこれだけ！
    「誰として」「何を」「どんな形で」
- サブテキスト：
    → テンプレートは全レクチャー分を用意済み
- ビジュアル要素：中央に大きく3語を横並び表示。下部に「テンプレート用意済み」のアイコン（ダウンロードマーク）
- 表示タイミング：0:40〜1:00
- 講師のセリフ（要旨）：「"誰として・何を・どんな形で"、この3つでOK。テンプレートは用意してあるので、次のセクションでさっそく使いましょう。」
```

## ⑥ 講師メモ（撮影時の注意点）

### トーンの指示
- テンションは「軽く・速く・自信を持って」。60秒しかないため、間を取りすぎない。ただし早口にはならず、1文ずつ明確に区切って話す
- 「難しいことを教える先生」ではなく「便利な裏ワザを教える先輩」の語り口

### NGワード

| NGワード | 言い換え |
|---------|---------|
| プロンプトエンジニアリング | AIへの指示の書き方 |
| コンテキスト | 背景情報、状況 |
| ハルシネーション | AIの間違い、でたらめな回答 |
| パラメータ | 設定、条件 |
| フォーマット | 形式、見た目 |
| トークン | 文字数、分量 |

### カメラ・マイク
- 顔出しなし、スライド全画面表示＋ナレーション音声のみ
- マイクは通常収録環境と同一。BGMは不要（60秒で集中を切らさないため）

### 編集指示
- スライド1の①②③フェードインアニメーションは編集時に追加
- スライド間のトランジションはカット切り替え（ワイプ・フェード不要、テンポ重視）
- スライド2の「←役割」「←具体的な指示」「←出力形式」のラベルは、講師がその部分を読み上げるタイミングに合わせて1つずつ表示するアニメーションを編集で追加

### 前後のレクチャーとの接続
- **冒頭（レクチャー2.4からの接続）：** 「準備は整いました」の一言で、アカウント作成・セキュリティ・社内許可の完了を暗示。2.4の内容を振り返る時間はない（60秒のため）ので、一言で済ませる
- **末尾（レクチャー3.1への橋渡し）：** 「次のセクションで、いよいよ実践です」と予告。S3の最初のレクチャーでPDFを実際にアップロードして3つの型を体験する期待感を持たせて終わる
