## ① レクチャーの位置づけ

前のレクチャー2.1で、受講者はClaudeとChatGPTのアカウントを作成し、ファイルをアップロードして最初の指示を送るところまで完了している。その直後に湧く「これ、お金かかるの？」「無料だとどこまでできるの？」という不安を、このレクチャーで即座に解消する。ゴールは**「無料プランのまま講座を進めて大丈夫だと確信できること」**の1点。課金への不安を取り除くことで、S3以降の実践パートに迷いなく進める状態を作る。次のレクチャー2.3では「会社のデータをAIに入れて大丈夫か」というセキュリティの不安に進むため、本レクチャーで「お金の不安」を完全に片付けておく橋渡しの役割を担う。

## ⑥ 講師メモ（撮影時の注意点）

### トーンの指示
- **「安心させる先輩」のトーン。** 受講者は「無料で本当に大丈夫なのか」「あとで高額請求が来ないか」という不安を抱えている。断言口調で安心感を与える。「大丈夫です」「十分です」「心配いりません」を迷いなく言い切る。
- テンションは落ち着いたトーン。煽らない。「有料プランがすごい」という方向に誘導しない。あくまで「無料で十分」を軸にする。

### NGワード

| NGワード | 言い換え |
|---------|---------|
| サブスクリプション | 月額プラン |
| トークン / トークン数 | 送れるメッセージの回数 |
| レート制限 / レートリミット | 1日に使える回数の上限 |
| GPT-4o / Claude 3.5 Sonnet 等のモデル名 | 「賢いモデル」「高性能なモデル」（受講者にモデル名の区別は不要） |
| ティア / Tier | プラン |
| コンテキストウィンドウ | 一度に読める量 |
| API | （このレクチャーでは言及しない） |
| 「課金すべき」「有料がおすすめ」 | 「無料で十分」「効果を実感してからで大丈夫」 |

### カメラ・マイク
- **画面共有のみ（顔出し不要）。** スライド5枚を順番に映す構成のため、顔出しワイプも不要。
- マイク音質のみ注意。落ち着いた声のトーンが重要なレクチャーのため、ノイズ除去を確認。

### 編集指示
- **カット・早送りは不要。** スライド切り替えのみの2分構成のため、基本的にワンテイク撮影で完成。
- スライド切り替え時に0.5秒の暗転トランジションを入れると、スマホ視聴時に切り替わりが見やすい。
- 各スライドの「この講座の内容 → 全部できる」の行が表示される瞬間に、軽い効果音（ポン、という肯定音）を入れると安心感が強まる。必須ではないがあると良い。

### 前後のレクチャーとの接続
- **冒頭（0:00〜0:05）：** 前のレクチャー2.1に軽く触れる。「アカウントを作ったところで」の一言で十分。2.1の内容を要約し直す必要はない。
- **末尾（1:50〜2:00）：** 次のレクチャー2.3を明確に予告する。「次はもう1つの不安、"会社のデータをAIに入れて大丈夫？"を解消します」と具体的に言う。「次のレクチャーもぜひご覧ください」のような曖昧な予告はNG。
