## ① レクチャーの位置づけ

本レクチャーはコース全7セクション・全15レクチャーの最終レクチャーであり、S6（DM一括生成）までに習得した全スキルを「明日から使い続ける仕組み」に変換するクロージングパートである。前のレクチャー6.3（1社ごとのパーソナライズDM）で受講者は「量産と質の両立」を体験しており、スキルとしては十分に揃っている。しかし、Udemy講座の最大の課題は「受講して満足→実務で使わない」という行動断絶である。本レクチャーのゴールは**「明日の朝、出社直後にやる"たった1つの行動"を決めて、そのための道具（特典テンプレート）をダウンロード済みの状態にすること」**の1点に絞る。次のレクチャーは存在しないため、橋渡しの代わりに「毎週月曜の朝10分」という最小習慣の提案で、コース外の実務への接続を担う。

## ⑥ 講師メモ（撮影時の注意点）

### トーンの指示
- 最終レクチャーのため、テンションは「熱く盛り上げる」ではなく**「静かに背中を押す」**。コーチが選手に最後の声をかけるイメージ
- 「お疲れさまでした」は冒頭の1回のみ。連発しない。ねぎらいよりも「明日やること」にフォーカスする
- 最後の20秒（スライド5）は特にスピードを落とし、語りかけるトーンで。間（ま）を怖がらない

### NGワード

| NGワード | 言い換え |
|---------|---------|
| API | （本レクチャーでは不使用。言及不要） |
| データポリシー | データの扱い方のルール |
| ワークフロー | 作業の流れ |
| テンプレート（特典名以外での多用） | ひな形、コピペ用の文面 |
| PDCAを回す | 毎週少しずつ試す |
| コミット | 決める、やると決める |
| リテラシー | 使いこなす力 |
| ナレッジ | コツ、ノウハウ |

### カメラ・マイク
- **顔出しなし。** 全編スライド表示＋ナレーション（声のみ）
- マイクは他レクチャーと同一環境。BGMなし（最後のスライド5のみ、極めて小さい音量のBGMを入れることは可。ただし主張しない音色に限る）

### 編集指示
- 本レクチャーは2分と短いため、カット・早送りは原則不要。一発撮りで収録する
- スライド切り替えのタイミングは、講師のセリフの「間」に合わせて0.5秒のフェード切り替えを入れる
- スライド3（ダウンロード手順）の表示中に、赤枠部分を拡大ズームする編集を入れる（スマホ視聴者が位置を視認できるようにするため）
- 最後のスライド5は、表示後2秒間の静止（無音）を入れてから終了する。余韻を持たせる

### 前後のレクチャーとの接続
- **冒頭（前のレクチャーとの接続）：** 「お疲れさまでした」の一言で十分。S6の内容を振り返らない。振り返りは受講者が最も離脱しやすいパターンのため、すぐに「明日やること」に入る
- **末尾（コース全体の締め）：** 次のレクチャーは存在しないため予告不要。「お疲れさまでした」で締めるが、これは冒頭とは異なり「一緒に走り終えた」ニュアンスで。受講後レビューの依頼は入れない（レビュー依頼はUdemyのシステム側で自動表示されるため、レクチャー内で言及すると蛇足になる）
